# git常用命令
`git init`:初始化仓库

`git clone`:克隆仓库

`git add`:暂存文件

`git status`:查看仓库状态

`git commit`:提交修改

`git reset`:回退版本(和一个reset不同噢，一个reset是清屏)

`git rm`:删除文件

`git mv`:移动/重命名文件

`git log`:查看提交历史

`git pull/push`:下载/上传远程仓库并合并
